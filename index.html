Timesheet CSV to PDF Filler

Upload a CSV file with the format: "date,time_in-time_out,num_hours"

Instructions:

  1. Prepare your CSV file with the format: date,time_range,hours
  2. Use 'X' for days off (e.g., '9/1/24,X,X')
  3. Time range format: 'start_time - end_time' (e.g., '11:15 - 5:00')
  4. Hours should be decimal numbers (e.g., '5.75')
  5. Upload the CSV file using the file input above
  6. Click the "Process CSV and Fill PDF" button
  7. Wait for the filled PDF to download
