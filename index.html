Timesheet Filler

Upload your input file and PDF template to generate a filled timesheet.

Instructions:

  1. Prepare your input file (CSV, TXT, or XLSX) with format: date,time_range,hours (e.g., '9/1/24,11:15 - 5:00,5.75')
  2. Use 'X' for days off (e.g., '9/1/24,X,X')
  3. Add your name as the last line of the input file
  4. Upload your input file in the first input
  5. Upload the Timesheet-Fillable.pdf in the second input
  6. Select the desired output format (PDF, PNG, or JPEG)
  7. Click "Process and Fill Timesheet"
  8. Download the filled timesheet when ready
